# SQL Data Analysis

## Objective
Analyze an e-commerce dataset using SQL queries.

## Tools Used
- MySQL Workbench
- SQL

## Tasks Performed
- SELECT Queries
- WHERE Filtering
- ORDER BY Sorting
- GROUP BY Analysis
- COUNT, SUM, AVG Aggregations

## Key Insights
- Analyzed 1200 e-commerce orders.
- Identified trends in order status, payment methods, and referral sources.
- Extracted business insights using SQL aggregations and grouping.
